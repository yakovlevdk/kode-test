# KODE Trainee — Тестовое задание

**ФИО:** Яковлев Денис Константинович
тг @dkyakovlev

## Декомпозиция

1. **Фаза 0:** Инициализация — Vite React-TS, зависимости, структура папок
2. **Фаза 1:** API и типы — fetchUsers, User, Department, маппинг департаментов
3. **Фаза 2:** Store (Effector) — события, стейты, эффекты, производные
4. **Фаза 3:** Top App Bar — поиск, сортировка, вкладки департаментов
5. **Фаза 4:** Список — UserCard, сортировка по алфавиту/ДР, пустой поиск
6. **Фаза 5:** Loading и ошибки — skeleton, критическая ошибка
7. **Фаза 6:** Страница детали — роут /user/:id, хедер, контакты
8. **Фаза 7:** Серверная фильтрация и кэш 5 мин
9. **Фаза 8:** Сохранение состояния (search, sort, department)
10. **Фаза 9:** Offline — баннеры ошибки/загрузки сети
11. **Фаза 10:** Тёмная тема, i18n, GitHub Pages
